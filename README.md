# Inventory & Order Management System

A production-ready assessment project for managing products, customers, orders, and inventory tracking.

## Stack

- Backend: Python, FastAPI, SQLAlchemy
- Frontend: React, Vite
- Database: PostgreSQL
- Containerization: Docker and Docker Compose

## Project Layout

```text
backend/     FastAPI API service
frontend/    React client application
docker-compose.yml
.env.example
```

Detailed setup, API, Docker, and deployment instructions will be added as the project is completed.
